- TWAS ANALYSIS

Perform TWAS analysis using FUSION (http://gusevlab.org/projects/fusion/) employing two data-set: H3K27ac peak and DNA methylation data ( data from http://eqtl.uchicago.edu/jointLCL/).

1.  Compute functional weights for each locus. We used FUSION default setting here: 5 folds of cross-validation, using 0.01 as minimum heritability p-value for which to compute weights. We employed LASSO and Elastic-net regression (with mixing parameter of 0.5) in this part.

2.  Do typical TWAS analysis. It takes pre-computed weights together with disease GWAS summary statistics to estimate the association of each locus to diseases.


We show the results below. Find significant peaks (whose TWAS.P.Bonferroni < 0.05 ). Then, select those in the significant gene area ([gene start position - 100kb, gene end position +100kb]). Significant genes are from Expression-TWAS results.
